# IMDb Movie Reviews - Sentiment Analysis

## Project Overview

This project performs Sentiment Analysis on IMDb movie reviews using Natural Language Processing (NLP) techniques. The objective is to classify reviews as Positive, Negative, or Neutral based on the sentiment expressed in the review text.

## Dataset

* Dataset: IMDb Movie Reviews Dataset
* Source: Kaggle
* Records Used: 1000 Reviews
* Features:

  * review
  * sentiment

## Tools and Libraries Used

* Python
* Pandas
* TextBlob
* Matplotlib
* Seaborn

## Project Workflow

1. Load IMDb movie review dataset.
2. Perform sentiment analysis using TextBlob.
3. Calculate sentiment polarity.
4. Classify reviews as:

   * Positive
   * Negative
   * Neutral
5. Visualize sentiment distribution using charts.
6. Save results to a CSV file.

## Visualizations

### 1. Sentiment Distribution Pie Chart

Displays the percentage of Positive, Negative, and Neutral reviews.

### 2. Sentiment Analysis Bar Chart

Shows the count of reviews in each sentiment category.

## Output Files

* sentiment_pie_chart.png
* sentiment_bar_chart.png
* sentiment_results.csv

## Key Insights

* Positive and Negative sentiments were successfully identified from review text.
* Sentiment polarity helped categorize customer opinions.
* Data visualization made sentiment patterns easy to understand.
* NLP techniques can be used to analyze large volumes of textual data efficiently.

## Conclusion

This project demonstrates how Natural Language Processing can be applied to real-world text data for sentiment classification. The results provide valuable insights into customer opinions and review trends.
